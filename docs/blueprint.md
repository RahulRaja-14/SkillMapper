# **App Name**: SkillMapper

## Core Features:

- Resume Skill Extraction: Resume parsing and skill extraction using AI to identify skills listed in the resume.
- Job Skill Extraction: Job description parsing and skill requirement extraction using AI to identify required skills. This feature will use a tool to assess if skills like communication or time management are indirectly described and required.
- Skills Matching: Comparison of extracted resume skills against required job skills. This feature will also list any lacking skills.
- Resource Suggestion: AI-powered suggestions for websites and YouTube channels to help users learn missing skills. The tool will consider the format of content best matched to the skill.
- Skill Report Display: Clear display of matched, missing and suggested skills, highlighting areas needing improvement.
- Data Input: Input fields for users to upload their resume and paste the job description.
- Client side: All processing happens client side

## Style Guidelines:

- Primary color: Dark teal (#468499) to convey professionalism and trust.
- Background color: Off-white (#F4F4F4) for a clean, modern feel.
- Accent color: Soft orange (#D97A4A) to highlight key information and calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, ensuring readability and modern aesthetics.
- Simple, clear icons to represent skills, suggestions, and report sections.
- Clean, well-spaced layout with clear sections for resume input, job description input, skills report, and suggestions.
- Subtle animations when generating skills report and suggestions.
